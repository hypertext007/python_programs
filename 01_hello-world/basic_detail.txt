question 1:how to add comment in program
so two type of comments method are avaliable.
1. single line comment.
2. multi line comment.

declared single line comment :-
# (string begins) is use to add single line comment in py.
ex :-
# how are you

declared multi line comment :-
'''(multi line string) ''' is use to add multi line comment in py.
ex :-
'''
user name - xyz
location - mars
'''
